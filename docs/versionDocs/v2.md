# v2 - JWT를 실제 웹 요청 기반으로 동작하게 만들기

## v2 구조
```scss
src/main/java/com/lab/jwtmvc/
├── controller/
│   └── AuthController.java        // /login, /refresh, /secure 엔드포인트
├── service/
│   └── AuthService.java           // 비즈니스 로직 (v1 재사용)
├── util/
│   └── JwtUtil.java               // JWT 생성/검증 로직 (v1 그대로)
├── store/
│   └── RefreshStore.java          // RefreshToken 저장소 (싱글톤 유지)
├── model/
│   ├── User.java
│   ├── AuthTokens.java
│   └── RefreshTokenRecord.java
└── JwtMvcApplication.java         // 스프링부트 실행 진입점
```
